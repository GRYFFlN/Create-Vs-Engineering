# Create: VS Engineering — Issue Tracker

A lightweight Create addon that adds a customizable **Ship Bearing** with a configurable reference angle and independent left/right rotation limits — so turrets, hinged mechanisms, and ship-mounted structures behave predictably without over-rotation or collision issues.

📦 [Modrinth](https://modrinth.com/mod/create-vs-engineering)

---

## 🐛 Reporting an Issue

Before opening a new issue, please:

1. **Search existing issues** to avoid duplicates.
2. **Update to the latest version** of the mod and check if the problem still occurs.
3. **Gather information** — Minecraft version, mod version, mod loader, and a crash report or log if applicable.

Then click **New Issue** and choose the appropriate template.
